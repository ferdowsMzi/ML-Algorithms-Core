# ML-Algorithms-Core

This repository is a collection of fundamental machine learning algorithms built completely **from the ground up**, without the use of high-level external libraries.

The goal of this project is to demonstrate a deep, **under-the-hood understanding** of the *mathematics* and *logic* driving these algorithms. It includes object-oriented implementations of ML algorithms like KNN, Decision Trees, Bayesian Networks, etc., complete with custom methods for feature selection, distance metrics, activation functions, and foundational mathematics.
